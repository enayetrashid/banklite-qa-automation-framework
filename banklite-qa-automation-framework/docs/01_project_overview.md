# Project Overview

## Purpose
This QA Automation Framework validates the BankLite Application using a structured testing approach.

## Scope
- API testing
- UI testing
- Manual and automated testing

## System Under Test
A demo banking application providing API and web interfaces.

## Approach
- Requirements → Acceptance Criteria → Test Scenarios → Test Cases
- Postman for API validation
- pytest for API automation (planned)
- Playwright for UI automation (planned)
